Upload the Resource Files on SAP Cloud Platform

You upload the ZIP files containing the extension application resource files on SAP HANA Cloud Platform using the SAP Cloud Platform console client.

Prerequisites

Note

This is a beta feature available for SAP Cloud Platform extension accounts. For more information about the beta features, see Using Beta Features in Accounts.

  • You have the ZIP file containing the extension application resource file on your local drive.

  • You have downloaded and configured SAP Cloud Platform console client. For more information, see Setting Up the Console Client.

Context

Using the upload-resource console client command, you upload ZIP files containing the extension application resource files, such as the SAP SuccessFactors HCM Suite role definitions and the Employee Central home page tile definitions, into the SAP Cloud Platform system repository of your extension account. You can share the the uploaded resources between the extension applications deployed in this account.

Procedure

  1. Open the command prompt and navigate to the folder containing neo.bat/sh (SDK installation folder/tools).
  2. To upload the ZIP file containing the resource files, execute the following command:
    neo upload-resource --name <resource_name> --location <path_to_resource> --account <account_name> --user <e-mail_or_user> --host <landscape_host>
  3. Enter your password if requested.
  4. Press ENTER and the upload of the file will start.
    If the ZIP file contains resources that already exist in the repository, the system notifies you and asks you whether to overwrite them. If the upload fails, check if you have defined the parameters correctly.

Results

You have uploaded the ZIP file containing the resources required for your extension application into the SAP Cloud Platform system repository for your extension account.
