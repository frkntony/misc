Subscribing to the SAP River RDE

To use SAP River RDE, you need to subscribe to the rde application.

Procedure

  1. Log on with a user (who is an account member) to the SAP Cloud Platform cockpit, using the developer account at https://account.hanatrial.ondemand.com/cockpit.
  2. Choose the Subscriptions section.
  3. Under Subscribed HTML5 Applications, choose the New Subscription... link.
  4. Select the rde application from the rde provider account.
  5. Choose Create.
