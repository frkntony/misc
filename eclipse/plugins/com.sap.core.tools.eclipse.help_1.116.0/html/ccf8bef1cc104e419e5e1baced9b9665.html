Managing Application Bindings

Access SAP Cloud Platform Enterprise Messaging (Beta) in SAP Cloud Platform cockpit to create and manage application bindings to messaging hosts.

Context

To complete the messaging setup, it is important to create application bindings that connect Java applications to messaging hosts. After an application binding has been created, applications can publish messages to any queue in the messaging host or subscribe to receive messages that have been published to any queue in the messaging host.

You can use the following procedure to create and manage an application binding.

Procedure

  1. Log on to SAP Cloud Platform cockpit and select an account.
  2. Choose Services in the navigation area. You can see the list of services available to you.

    For more information about accessing services, see Services in the Cockpit.

  3. Choose Enterprise Messaging.
  4. Choose Application Bindings in the navigation area.

    You can see a list of existing application bindings.

  5. To create an application binding, perform the following substeps:
    1. Choose Create Application Binding.
      Note At least one Java application must be available to create an application binding. For more information on developing Java applications, see Developing Java Applications.
    2. Select a Java application.
    3. Select the messaging host to which you want to bind the application.
    4. Enter a name for the application binding.
      Note This name must be unique across all application bindings associated with the selected Java application.
    5. Choose Save.
  6. To delete an application binding, choose (Delete Binding).
