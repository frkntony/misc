Application Runtime Container

SAP Cloud Platform applications run on a modular and lightweight application runtime container where they can use the platform services APIs and Java EE APIs according to standard patterns.

Depending on the runtime type and corresponding SDK you are using, SAP Cloud Platform provides the following profiles of the application runtime container:
Profile Provides support for Supported Java versions Use

Java Web

Some of the standard Java EE 6 APIs (Servlet, JSP, EL, Websocket)

7 (default); 6

If you need a small standalone Java Web container.

Java Web Tomcat 7

Some of the standard Java EE 6 APIs (Servlet, JSP, EL, Websocket)

7 (default); 8

If you need a simplified Java Web application runtime container based on Apache Tomcat 7.

Java EE 6 Web Profile

Java EE 6 Web Profile APIs

7 (default); 6

If you need an application runtime container together with all containers defined by the Java EE 6 Web Profile specification.

Java Web Tomcat 8

Some of the standard Java EE 7 APIs (Servlet, JSP, EL, Websocket)

8 (default); 7

If you need a simplified Java Web application runtime container based on Apache Tomcat 8.

For the complete list of supported APIs, see Supported Java APIs
