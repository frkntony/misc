Exporting Apps

Prerequisites

You have the role TenantOperator, are logged into the gamification workbench and have opened the Operations tab, and navigated to the Data Management section.

Context

The gamification service allows exporting all available apps including their content. You can choose between a full tenant export including all player data and an export of game mechanics only. The latter can be imported again.

Procedure

  1. Select the Export mode in the combo box labeled Export in the form area Import / Export.

    • Full Export: export all game mechanics and player data.

    • Game Mechanics: export game mechanics only.

  2. Press Download to start the export. Your browser should show the file storing dialog.
  3. Store the provided ZIP file on your disk.
