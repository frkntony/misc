(Optional) List the Resources Available in the System Repository

You can check whether the resource file containing the role definitions for your application is available in the system repository for the account where the application is deployed.

Prerequisites

Note

This is a beta feature available for SAP Cloud Platform extension accounts. For more information about the beta features, see Using Beta Features in Accounts.

  • You have uploaded the ZIP file containing the SAP SuccessFactors HCM suite role definitions in the system repository for the account where your extension application is deployed.

  • You have downloaded and configured SAP Cloud Platform console client. For more information, see Setting Up the Console Client.

Context

Using the list-resources console client command, you list the names of the extension application resource files available in the system repository for a specified extension account.

Procedure

  1. Open the command prompt and navigate to the folder containing neo.bat/sh (SDK installation folder/tools).
  2. To list the extension application resource files available in the system repository for the specified account, execute the following command:
    neo list-resources --account <account_name> --user <e-mail_or_user> --host <landscape_host>
  3. Enter your password if requested.
  4. Press ENTER.

Results

The console client lists all the extension application resource files available in the SAP Cloud Platform system repository for the specified account.
