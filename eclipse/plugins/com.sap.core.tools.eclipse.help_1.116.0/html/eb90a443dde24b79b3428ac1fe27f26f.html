HelpDesk App - Standalone Deployment

Prerequisites

You have an Eclipse IDE with SAP Cloud Platform tools.

For more information about how to install the SAP Cloud Platform tools, see Eclipse Tools.

Context

The demo application's (Help Desk) source code is also available in GitHub.

This section explains how to set up an Eclipse project, deploy the demo application on SAP Cloud Platform, and configure it to run with your gamification service subscription.

Procedure

  1. Download sources as a zip from GitHub.

  2. Extract the archive.
  3. Open Eclipse with SAP Cloud Platform tools and choose  File  Import .
  4. Choose  Maven  Existing Maven Projects .

  5. Choose the folder containing the demo application sources and choose Finish.
  6. Deploy and start the demo application on the cloud from Eclipse IDE. Select Java Web as a Runtime.

    For more information, see Deploying on the Cloud from Eclipse IDE.

  7. Configure destinations and roles for the deployed application. Use the same configuration as described in section HelpDesk App - Configuration of Available Subscription.
