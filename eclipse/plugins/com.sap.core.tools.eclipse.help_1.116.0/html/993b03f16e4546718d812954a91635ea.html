Administering Application Feedback

Context

As a feedback administrator, you can add applications and administer applications' feedback.

Procedure

  1. Open the feedback Administration UI.
  2. Add an application by choosing the +Add button and enter a name for the application for which feedback is to be collected.
  3. To customize the description of a rating or free text question, click on the pencil icon in the respective question row.
  4. To customize the description of a context attribute, repeat Step 3.
  5. To free up quota space, click on the Free Up Quota Space link and choose a specific time period for which feedback data is to be permanently deleted.
