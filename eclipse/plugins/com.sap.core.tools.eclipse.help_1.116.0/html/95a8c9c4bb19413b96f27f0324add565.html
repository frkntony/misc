Prerequisites for Translating ABAP Short Texts Using an RFC

To use SAP Translation Hub to translate short texts that you create in ABAP systems, several prerequisites are required.

Prerequisites
Topic Description
Source language The source language of texts that you want to translate must be English.
Cloud connector Your cloud connector must be configured to make your ABAP system reachable. In particular, you must grant access to RFCs with the LXE_STH prefix.

For more information, see Limiting the Accessible Resources for RFC in Configuring Access Control (RFC).

RFC destination There must be a remote function call (RFC) destination to your ABAP system. The destination name must follow this pattern:

STH_<name of the current HCP account>

Note

You can take your HCP account name from your HCP landscape URL. It appears after /acc, as indicated in the following sample URL:

https://account.hanatrial.ondemand.com/cockpit#/acc/<HCP account name>/

For more information, see Configuring Destinations from the Cockpit.
RFC-enabled function modules To enable SAP Translation Hub to read texts in development objects in your ABAP system, RFC-enabled function modules are required. To create these function modules in your ABAP system, implement SAP Note 2349776 .
Authorizations in ABAP system

To enable the communication with the ABAP system, you need a technical user of type Communications Data in the ABAP system. This user requires generic authorizations and authorizations that are specific to SAP Translation Hub.

Generic authorizations:
  • Authorization object S_RFC
    For details, see the section on calling a function module using the dynamic repositories (SAP Java Connector, JCo), in SAP Note 460089 .
    Note Since there are differences between the various releases supported, make sure that you use the settings for the release of your ABAP system.
Authorizations specific to SAP Translation Hub:
  • Authorization object S_RFC with the following attributes:
    • ACTVT: 16 (Execute)
    • RFC_TYPE: FUNC (Function Module)
    • RFC_NAME: LXE_STH_GET_OBJECT_LISTS, LXE_STH_READ_OBJECT_LIST, LXE_STH_READ_TEXTS, LXE_STH_WRITE_TEXTS
  • Authorization object S_ADMI_FCD with the following attributes:
    • Field name S_ADMI_FCD (System Administration Function)
    • Activity TRNR (Translation Administration SLWA/SLWB)
Object lists for translation You have generated object lists in the ABAP system. For information about object lists and the translation setup in ABAP systems, see SAP Help Portal at http://help.sap.com/nw under  SAP NetWeaver Platform  <your NetWeaver version>  Application Help  Function-Oriented View . Then follow the path listed below for your version of SAP NetWeaver:

     Application Server  Application Server ABAP  Other Services  Services for Information Developers and Translators  Setting Up and Coordinating Translation (BC-DOC-TTL) 

  • 7.0 EHP3:
