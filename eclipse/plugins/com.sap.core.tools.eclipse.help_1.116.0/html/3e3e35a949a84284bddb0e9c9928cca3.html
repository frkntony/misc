Connecting Remote Systems

You can setup HANA cloud connector and establish a connection to ABAP backend systems.

Context

For every system to connect perform the following steps:

Procedure

  1. Log on with a user (who is an account member) to the SAP Cloud Platform cockpit to the developer account at https://account.hanatrial.ondemand.com/cockpit.
  2. Create the following new destinations which target the logical system on the cloud connector with the following general settings:
    • For OData access with the URL path /sap/opu/odata.

      Add the additional property RDEUsage odata_abap.

    • With path /sap/bc/ui5_ui5.

      Add the additional property RDEUsage ui5_execute_abap.

    • Only for 7.30 with UIAddOn or 7.40 ABAP systems with path /sap/bc/adt.

      Add the additional property RDEUsage dev_abap.

  3. For all destinations apply the following general settings:
    • Name can be chosen freely but should indicate the system.
    • Description can be chosen freely but should indicate the system and the usage.
    • Proxy type OnPremise
    • Specify an additional property RDEEnabled with value true.
    • Specify an additional property RDESystem with value <system_ID> which is the system ID of the ABAP system.
