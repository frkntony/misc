Supported SAP HANA Web-based Tools

SAP Cloud Platform supports the following Web-based tools: SAP HANA Web-based Development Workbench, SAP HANA Studio, and SAP HANA XS Administration Tool.

Prerequisites

  • You have a database user. See Guidelines for Creating Database Users.
  • Your database user is assigned the roles required for the relevant tool. See Roles Required for Web-based Tools.

Context

You can access the SAP HANA Web-based tools using the Cockpit or the tool URLs. The following table summarizes what each supported tool does, and how to acess it.

Table 1: Supported Web-Based Tools for SAP HANA Development and Administration

Description

SAP HANA Web-based Development Workbench

Includes an all-purpose editor tool that enables you to maintain and run design-time objects in the SAP HANA repository. It does not support modeling activities.

See Developer Guide for SAP HANA Web Workbench for the latest release of SAP HANA or Developer Guides for earlier releases of SAP HANA.

Development Tools section: SAP HANA Web-based Development Workbench

https://<database instance><account>.<landscape host>/sap/hana/xs/ide/

SAP HANA Cockpit

Provides you with a single point-of -access to a range of Web-based applications for the online administration of SAP HANA.

See Administration Guide for the latest release of SAP HANA or Administration Guides for earlier releases of SAP HANA.

Note It is not possible to use the SAP HANA database lifecycle manager (HDBLCM) with the cockpit.

Administration Tools section: SAP HANA Cockpit

https://<database instance><account>.<landscape host>/sap/hana/xs/admin/cockpit

SAP HANA XS Administration Tool

Allows you, for example, to configure security options and HTTP destinations.

See Administration Guide for the latest release of SAP HANA or Administration Guides for earlier releases of SAP HANA.

Administration Tools section: SAP HANA XS Administration

https://<database instance><account>.<landscape host>/sap/hana/xs/admin/
Remember

When using the tools, log on with your database user (not your SAP Cloud Platform user). If this is your initial logon, you will be prompted to change your password. You are responsible for choosing a strong password and keeping it secure.
