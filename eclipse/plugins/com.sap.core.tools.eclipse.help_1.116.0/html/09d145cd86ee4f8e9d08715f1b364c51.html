Quality

SAPUI5 supports several testing frameworks, such as QUnit testing and One Page Acceptance tests.

This section describes the exact procedures needed to set up specific testing frameworks for the SAPUI5 environment, as well as some guiding principles and best practices, for example relating to QUnit testing and One Page Acceptance (OPA) tests. Furthermore, this section also contains a list of do's and don'ts and a cookbook for writing SAPUI5 controls.
