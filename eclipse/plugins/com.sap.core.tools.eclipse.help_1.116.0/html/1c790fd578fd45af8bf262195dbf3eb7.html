Setting Up Eclipse

Before developing your application, you need to download and set up the Eclipse IDE for Java EE Developers.

Procedure

  1. Install Eclipse.
  2. Set up Git in Eclipse.
    1. To configure your user settings, choose  Window  Preferences  Team  Git  Configuration .
    2. To add your email address and your name, choose Add Entry... and enter the following key-value pairs:
      Key Value
      user.email <e-mail address of SCN user>
      user.name <Firstname> <Lastname>
