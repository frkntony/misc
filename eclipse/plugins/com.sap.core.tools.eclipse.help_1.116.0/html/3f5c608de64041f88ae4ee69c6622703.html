General Coding Guidelines (internal)

The following guidelines are valid in SAPUI5 and apply controls that are shipped with SAPUI5:

  • The naming conventions for SAPUI5 controls are mandatory for SAP internal development, see Naming Conventions for Control and Application Development

  • Adhere to the JavaScript development guidelines for SAPUI5 development available in Trac, see Development Guidelines.

  • Do not create global variables; in control behavior you always use this variable for storing instance data or sap.ui.<libname>.<controlname> for storing static data for the control class.

  • If you need a DOM element, use jQuery.sap.domById(id) instead of document.getElementById(id); this fixes some browser bugs.

  • Use jQuery.sap.byId(id) instead of jQuery("#" + id) if you need a jQuery object (because this also does the necessary escaping of the ID).
