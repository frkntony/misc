Hello World Tutorial Using Eclipse

This tutorial illustrates how to build a simple HTML5 application.

Prerequisites

  • You have set up Eclipse, see Installing Eclipse IDE.
  • Your company has signed up for an SAP Cloud Platform account.
  • You are a member of the SAP Cloud Platform account.

Context

During the tutorial, you perform the following tasks:
  1. Creating an HTML5 Application
  2. Cloning a Repository
  3. Creating a Project and Adding an HTML File
  4. Pushing a File to the Git Repository
  5. Testing the Application
  6. Creating a Version
  7. Activating a Version
