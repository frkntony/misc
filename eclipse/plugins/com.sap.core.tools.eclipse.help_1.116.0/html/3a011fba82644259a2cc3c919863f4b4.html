Building Base URL of SAP Translation Hub

The base URL is main part of the complete URL that you need to access the services of SAP Translation Hub.

To access the services and user interface of SAP Translation Hub on SAP Cloud Platform, you require the base URL plus additional parameters that are specific to the individual services and features in SAP Translation Hub.

The following table summarizes the parts of the base URL:

Landscape Base URL Sample Base URL

Trial

 https://saptranslation-<account name on HCP>.hanatrial.ondemand.com/ https://saptranslation-s0001001002trial.hanatrial.ondemand.com/

Productive

 https://sap<provider account name>-<subscription account name on HCP>.<landscape host>/ https://sapa7d6108fa-c77539gg5d.hana.ondemand.com/
Note For details about the name of the service and the query parameters that you add to the base URL, see the documentation for the individual services.
