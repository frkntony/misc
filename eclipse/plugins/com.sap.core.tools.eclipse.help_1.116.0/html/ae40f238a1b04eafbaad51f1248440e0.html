Managing Members

Added information about new features in the member list:

The names of members are now displayed in addition to their user IDs.

Note The name of a member is displayed only after the member visits the account for the first time.

Administrators can now send e-mails to members of the account.

Sending e-mails to members is only possible after the recipient has logged on to the account.
