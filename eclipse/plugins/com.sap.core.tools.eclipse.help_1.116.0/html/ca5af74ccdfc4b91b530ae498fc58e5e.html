Changing the UI Port

Context

By default, the Cloud connector uses port 8443 for its administration UI. In case this port is blocked by another process, or if you just want to change it after the installation (on Windows, you can choose a different port during installation), you can use the changeport tool, provided with Cloud connector version 2.6.0 and higher.

Procedure

  1. Change to the installation directory of the Cloud connector. To adjust the port, execute the following command:
    • Microsoft Windows OS:
      changeport <desired_port>
    • Linux OS, Mac OS X:
      ./changeport.sh <desired_port>
  2. The tool will inform you about the successful modification of the port.
  3. To activate the new port, you need to restart the Cloud connector.
