Managing Queues

Access SAP Cloud Platform Enterprise Messaging (Beta) in SAP Cloud Platform cockpit to manage queues in your messaging hosts.

Context

Messaging hosts use queues to enable point-to-point communication between two Java applications. A queue can be configured to deliver messages in different ways when multiple applications are connected to it. This property of a queue is called access type and it can be one of the following values:
  • Exclusive: Only one application can receive messages from an exclusive queue. This is typically the first application that connects to the queue.
  • Non-Exclusive: Multiple applications can connect to a non-exclusive queue. Each application is serviced in a round-robin fashion to provide load-balancing.

You can use the following procedure to access and manage the list of queues you have created for a specific messaging host:

Procedure

  1. Log on to SAP Cloud Platform cockpit and select an account.
  2. Choose Services in the navigation area. You can see the list of services available to you.

    For more information about accessing services, see Services in the Cockpit.

  3. Choose Enterprise Messaging.
  4. Choose Messaging Hosts in the navigation area.
  5. Select a messaging host by choosing the corresponding link.
  6. Choose Queues in the navigation area.
    You can see the following information:
    • List of queues created for the selected messaging host
    • Number of messages stored in each queue
    • Size (in kilobytes) of all the messages stored in each queue
    • Access type of each queue (exclusive or non-exclusive)

    You can filter the queues by selecting a specific access type.

    You can also search for a queue by typing its name in the Search text box. The list of queues is filtered to match the pattern you have entered.

  7. To create a new queue, perform the following substeps.
    1. Choose Create Queue.

      The Create New Queue dialog box appears.

    2. Enter a queue name.
    3. Select an access type.
    4. Choose Save.
  8. Manage the queues by performing one or more of the following administrative tasks:

    • Edit a queue to change its access type.

    • Clear all the messages in a queue.

    • Delete a queue.
