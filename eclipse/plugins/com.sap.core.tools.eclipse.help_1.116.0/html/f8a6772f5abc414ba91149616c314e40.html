Integration of SAP Translation Hub with SAP Hybris Cloud for Customer

The integration of SAP Translation Hub in SAP Hybris Cloud for Customer for translating user interface texts is an example of how to integrate SAP Translation Hub with other products.

You can use the Language Adaptation work center in SAP Cloud Platform for Customer to translate work center texts that are proposed by SAP Translation Hub.

Example

Assume that you use SAP Hybris Cloud for Customer and want to expand your operations to a country whose language is currently not supported by SAP Hybris Cloud for Customer. In this scenario, you can translate work center texts into one of the languages that SAP Translation Hub supports, but that SAP Hybris Cloud for Customer does not support as standard.
