Analyzing Gamification Concepts

Gamification is a continuous process. It is crucial to continuously monitor the influence of a gamification concept and react to the users' behavior. For example, you want to know if your gamification concept motivates the target group or if users lose interest.

The gamification service offers basic analytics: for example, the assignment of points or badges to users over time. Therefore, you can analyze peaks and troughs of user achievements.
