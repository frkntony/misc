Creating Destinations

To connect applications to other systems, you have to create destinations. They store all connection relevant parameters.

Context

The following destinations are required to access the used UI5 resources.

Procedure

  1. Log on with a user (who is an account member) to the SAP Cloud Platform cockpit, using the developer account URL at: https://account.hanatrial.ondemand.com/cockpit.
  2. Choose the Destinations section.
  3. To create a new destination for SAPUI5 resources, choose New Destination and enter the following data:
    Field Value
    Name ui5dist
    Type HTTP
    URL https://sapui5.hana.ondemand.com/sdk/resources
  4. To create a new destination for SAPUI5 test resources, choose New Destination and enter the following data:
    Field Value
    Name ui5dist-test-resources
    Type HTTP
    URL https://sapui5.hana.ondemand.com/sdk/test-resources
