list-subscribed-applications

Lists all Java applications to which a given account is subscribed 

neo list-subscribed-applications --account <account_name> --user <e-mail_or_user> --host <landscape host>
Parameters
To list all parameters available for this command, execute neo help list-subscribed applications in the command line.
Required
-a, --account Account name

This is the account of the applications consumer.

Type: string (up to 30 characters; lowercase letters and numbers, starting with a letter)
-u, --user Use your email, SAP ID or user name

To be able to execute this command, the specified user must be a member of the account.

Type: string

-h, --host Use the respective landscape host for your account type.

Type: URL. For acceptable values see Landscape Hosts
-p, --password To protect your password, enter it only when prompted by the console client and not explicitly as a parameter in the properties file or the command line.

Type: string

Example
neo list-subscribed-applications --account consumeraccount --user myuser --host hana.ondemand.com
