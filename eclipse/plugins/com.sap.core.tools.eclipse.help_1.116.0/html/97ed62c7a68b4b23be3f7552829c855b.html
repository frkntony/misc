Destination Parameters

This section lists the supported destination parameters.

MTA Parameter Type Mandatory Sensitive Possible Values Default Value Comment
name String Yes        
type Enum     HTTP, LDAP, MAIL, RFC HTTP  
description String          
url URL Yes       Use when the parameter type has the HTTP or LDAP values. Mandatory only for these types.
proxy-type Enum     Internet, OnPremise Internet Use with HTTP or LDAP destination type.
authentication Enum     NoAuthentication, BasicAuthentication, AppToAppSSO, ClientCertificateAuthentication, OAuth2SAMLBearerAssertion, PrincipalPropagation, SAPAssertionSSO NoAuthentication Use with HTTP or LDAP destination type.
user String         Use when:
  • if BasicAuthentication is the Use Authentication type
  • if the parameter type has the values MAIL, or RFC.
password String   Yes     Use when:
  • if BasicAuthentication is the Authentication type
  • if MAIL, or RFC are the destination type.
audience String Yes       Use when OAuth2SAMLBearerAssertion is the Authentication type. Mandatory only for this authentication.
client-key String Yes Yes     Use when OAuth2SAMLBearerAssertion is the Authentication type. Mandatory only for this authentication.
token-service-url URL Yes       Use when OAuth2SAMLBearerAssertion is the Authentication type. Mandatory only for this authentication.
token-service-user String   Yes     Use when OAuth2SAMLBearerAssertion is the Authentication type.
token-service-password String         Use when OAuth2SAMLBearerAssertion is the Authentication type.
system-user String         Use when OAuth2SAMLBearerAssertion is the Authentication type.
issuer-sid String         Use when SAPAssertionSSO is the Authentication type.
issuer-client String         Use when SAPAssertionSSO is the Authentication type.
recipient-sid String         Use when SAPAssertionSSO is the Authentication type.
recipient-client String         Use when SAPAssertionSSO is the Authentication type.
certificate String   Yes     Use when SAPAssertionSSO is the Authentication type.
signing-key String   Yes     Use when SAPAssertionSSO is the Authentication type.
system-user String         Use when OAuth2SAMLBearerAssertion or SAPAssertionSSO are the Authentication type.
repository-user String         Use with the RFC parameter type.
repository-password String   Yes     Use with the RFC parameter type.
jco-client String Yes   3 digits   Use with the RFC parameter type. Mandatory only for this authentication.
jco-ashost String     00-99   Use with the RFC parameter type.. Either this or jco-mshost must be specified.
jco-sysnr String         Use with the RFC parameter type, if jco-ashost is specified.
jco-mshost String         Use with the RFC parameter type. Either this or jco-ashost must be specified.
jco-r3name String     3 letters or digits   Use with the RFC parameter type, if jco-mshost is specified.
jco-msserv String         Use with the RFC parameter type., if jco-mshost is specified.
additional-properties Map         Use to define arbitrary additional properties.
Note In the case of account-level destinations, we recommend you place any sensitive parameters in the MTA extension descriptor. In the case of the com.sap.java module type, we recommend that you move all destination parameters to the extension descriptor.
