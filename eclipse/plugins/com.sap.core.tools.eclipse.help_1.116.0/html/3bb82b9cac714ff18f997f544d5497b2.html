Assigning Developers to a Repository

Permissions for Git repositories are granted based on the account member roles of users. To grant an account member access to a Git repository, assign one of these roles: Administrator, Developer, or Support User.

Prerequisites

  • You are an administrator of the account.

  • New members to be added already have SAP user IDs.

Context

For details about the permissions associated with the individual roles, see Security.

Procedure

  1. Log on to the SAP Cloud Platform cockpit.
  2. Assign members to accounts and define their roles (see Related Information).

    Make sure that you assign at least one of these roles: Administrator, Developer, or Support User.
