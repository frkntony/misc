Creating and Working with Translation Projects

With SAP Translation Hub special user interface (UI), you can create and manage translation projects.

When you create a translation project, the setup and translation steps to follow depend on the source of texts that you wat to translate. You can create translation projects for the following text repositories:
  • Properties files from the Git repository of SAP Cloud Platform
  • Short texts from ABAP systems using a remote function call (RFC)
  • Individual files that you upload directly to SAP Translation Hub's UI
