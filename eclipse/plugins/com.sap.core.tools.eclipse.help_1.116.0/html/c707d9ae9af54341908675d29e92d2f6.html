Consuming the Document Service (HTML5 Applications)

You can connect to the document service by treating it as an external service and the document service treats your HTML5 application as an external app that requests access.

Procedure

To enable external access to your document service repositories, deploy a small proxy application that is available out-of-the-box. For more information about its usage and deployment, see Accessing the Document Service from an HTML5 Application.
