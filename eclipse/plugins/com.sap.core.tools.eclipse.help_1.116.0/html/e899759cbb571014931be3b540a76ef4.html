Error Code 404

Symptom

The destination HTTP response has status code 404.

Problem

The related tunnel has never been created.

Keywords

error, 404, SAP Cloud Platform Connector, SCC

Solution

  1. In the Cloud connector, check whether the tunnel is operational and that it is connected to the correct account in SAP Cloud Platform. If not, connect the tunnel and try again.
  2. In the Cloud connector, reconfigure the tunnel settings:
    1. Open Settings.
    2. Go to the Configuration tab and configure Account, User, and Password.
    3. Go to HTTPS Proxy settings and configure them (if needed).
  3. If the problem persists, create a ticket in the SAP customer support system to describe your problem. For more information, see Connectivity Troubleshooting Guide â†’ SAP Support Information.
