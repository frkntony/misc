SAPUI5: Getting Started

Here you find information on how to adjust your environment and to get started with some easy examples including an introduction to the SAPUI5 tools.

Note

The following sections focus on building application in Eclipse. For further information on how to build applications with the Cloud-based tool SAP Web IDE, see the documentation for SAP Cloud Platform at https://help.hana.ondemand.com/ under  Platform Solutions  SAP Web IDE .

Read the important information under Read Me First. Check the list of supported browsers and platforms to choose those that fit your needs.

Sign up for an SAP Cloud Platform account. When having installed SAP Cloud Platform Tools, SAPUI5 runtime and tools are included and ready to run.
