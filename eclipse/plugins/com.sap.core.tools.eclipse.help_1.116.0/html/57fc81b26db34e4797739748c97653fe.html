Working with Repositories

Git supports a large variety of commands for working with repositories.

The following assumes that you are familiar with the concepts of Git and that you have access to a suitable Git client, for example SAP Web IDE, to perform Git operations.

If you are new to Git, we strongly recommend that you read a text book about Git and consult the Best Practices guide before using the Git service. The Troubleshooting Guide helps solve some common issues you may encounter when working with the Git service.
