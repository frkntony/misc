Eclipse Tools

SAP Cloud Platform Tools is a Java-based toolkit for Eclipse IDE. It enables you to perform the following operations in SAP Cloud Platform:

  • Develop and deploy Web applications
  • Configure connectivity destinations
  • View and configure logs and log files
  • Manage user roles and groups
  • Profile Web applications locally
  • Build and adapt Web applications using SAPUI5
Features

You can download SAP Cloud Platform Tools from the SAP Development Tools for Eclipse page. The toolkit package contains:

  • SAP JVM Tools
  • SAP Cloud Platform Tools for Java
  • Documentation for SAP Cloud Platform
  • UI development toolkit for HTML5 (Developer Edition)
Support

SAP Cloud Platform Tools come with a wizard for gathering support information in case you need help with a feature or operation (during deploying/debugging applications, logging, configurations, and so on). For more information, see Support Information (Eclipse IDE).
