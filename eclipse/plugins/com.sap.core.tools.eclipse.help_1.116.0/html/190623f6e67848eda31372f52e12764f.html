Updating Levels

Prerequisites

You have logged on to the gamification workbench with the role GamificationDesigner and you have opened the Levels tab.

Context

Procedure

  1. Select the level in the list to be updated.
  2. Press Edit.
  3. Change values of attributes that shall be updated.
  4. If a level has an image check Delete Image checkbox to remove the image.
  5. Press Save.
