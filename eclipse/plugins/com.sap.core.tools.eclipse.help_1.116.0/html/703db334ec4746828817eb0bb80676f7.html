Client API (HANA XS)

The SAP Cloud Platform document service (document service) is exposed using the OASIS standard protocol Content Management Interoperability Service (CMIS).

The CMIS standard defines the protocol level (SOAP, AtomPub, and JSON based protocols). The SAP HANA Cloud Platform provides a document service client API on top of this protocol for easier consumption. The JavaScript library supports the JSON protocol.
