Error Code 500

Symptom

The destination HTTP response has status code 500 and the HttpResponse body contains the following text:

Could not connect to the requested service

Problem

The connected Cloud connector has no access to the requested back-end service.

Keywords

error, 500, SAP Cloud Platform connector, SCC

Solution
  1. Check the configuration of the destinations and the Cloud connector: The URL of the destination used should consist of a host and a port that are configured on the Access Control tab page as virtual host and port.
  2. Check that the back-end system is running and accessible from the Cloud connector machine.
  3. Check that the correct protocol is used for the requested back-end system in the destination configuration (that is, HTTP/HTTPS), as it was specified on the Access Control tab.
  4. If a system certificate is configured, make sure it is the correct certificate with permissions to the relevant back-end system.
  5. If the problem persists, create a ticket in the SAP customer support system to describe your problem. For more information, see Connectivity Troubleshooting Guide â†’ SAP Support Information.
