Error Code 503

Symptom

The destination HTTP response has status code 503 and the HttpResponse body contains the following text:

No route for the requested service

Problem

The connectivity proxy in SAP Cloud Platform cannot resolve the requested URL. Either the URL of the destination used is incorrect, or the service is not configured for the connectivity proxy, or the tunnel is down.

Keywords

error, 503, SAP Cloud Platform connector, SCC

Solution
  1. Check the requested URL used by the destination (the host and port of the URL should match the specified virtual host and port in the Cloud connector).
  2. Check that the tunnel is established. The Cloud connector should indicate a Connected state for the relevant SAP Cloud Platform Platform account.
  3. Check that the back-end system required is configured on the Access Control tab page.
  4. If the problem persists, create a ticket in the SAP customer support system to describe your problem. For more information, see Connectivity Troubleshooting Guide â†’ SAP Support Information.
