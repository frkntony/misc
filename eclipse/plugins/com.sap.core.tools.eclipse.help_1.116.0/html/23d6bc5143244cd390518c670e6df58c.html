Installing and Configuring SAP Lumira Server on SAP Cloud Platform

This guide describes how you install SAP Lumira Server on your SAP HANA instance for your SAP Cloud Platform account, and how you configure it.

Overview

SAP Lumira Server is a solution that combines and visualizes data from multiple sources, thus allowing analysts and decision makers to use various visualizations to analyze data and collaborate with colleagues. You can use SAP Lumira Server to analyze data stored in SAP HANA database running on SAP Cloud Platform.

Remember

SAP Cloud Platform allows you to install and update SAP Lumira Server in an automated manner on top of SAP HANA instances within your account. After installing, you are responsible for configuring your SAP Lumira Server, creating visualizations and monitoring the availability.

To use the SAP Lumira Server on SAP Cloud Platform, you first need to install the components in SAP Cloud Platform cockpit and then to configure the server.

Note

If you wish to monitor the availability of individual visualizations, please follow the description on how to configure availability check for XS applications from the cockpit. SeeConfiguring Availability Checks for SAP HANA XS Applications from the Cockpit.

Installing SAP Lumira Server on SAP Cloud Platform

Procedure

  • You need to have a valid license for SAP Lumira Server to be able to request this feature to be enabled for your account at SAP Cloud Platform Platform.
  • Request SAP Lumira Server to be enabled for your account by creating a ticket in SAP support component BC-NEO-PERS. To request support for other issues or requests related to SAP Lumira Server, check the components described in SAP Note 1888290 .

Configuring SAP Lumira Server

After you install SAP Lumira Server on SAP Cloud Platform, implement some of the configuration steps described in SAP Lumira Server Installation Guide. Namely, these are as follows (each step below refers to a relevant section in the SAP Lumira Server Installation Guide):

Procedure

  1. (For SAP HANA SP9 or later only) Assign catalog privileges to roles as described in section 6.5.
  2. Continue with all other configuration steps as described consecutively in sections 6.6 through 7.3.
  3. Optionally, if you want to enable further features of SAP Lumira Server, such as chart extensions or currency conversion, you perform all other configuration steps from section 8 through the end of the guide.
  4. After configuring your SAP Lumira Server at SAP Cloud Platform, disable the XS jobs as described in section 6.1 of the SAP Lumira Server Installation Guide. This is necessary to prevent serious performance impact due to overload of HANA _SYS_XS.JOB_LOG and unnecessary back-up requests for your HANA system. This may lead to slower performance of the Related Visualizations feature in SAP Lumira Server, as they will be generated on the fly.

Updating SAP Lumira Server

Procedure

  1. Open the SAP Cloud Platform cockpit.
  2. Choose the Check for Updates link.

    If there are updates available for SAP Lumira Server, you will see them listed with details.

    Note

    For SAP Lumira Server updates, you usually have to update the Universal Analytics Functional library and SAP Lumira Server. Check relevant SAP Notes for additional info.

  3. Choose the required update and install it on your SAP HANA instance.
