Supported Languages

SAP Translation Hub supports a number of different languages. The following table shows the languages into which you can translate texts using the multilingual text repository (MLTR) and machine translation.

You use the language IDs in the table when you request translations with the SAP Translation Hub translation service.
Note The source language of texts that you want to translate must be English.

To get the latest list of languages, call <base URL of SAP Translation Hub>/translationhub/api/v1/languages.

For information about the base URL, see Building Base URL of SAP Translation Hub in the Related Information section.
