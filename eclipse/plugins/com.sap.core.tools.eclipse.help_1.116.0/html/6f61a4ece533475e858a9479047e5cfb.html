Access to Databases in Other Accounts

You want to access data from a productive SAP HANA or SAP ASE database in another account and you need the required permissions. The account prodiving the permission gives you access by providing you with a token and a database user, which you use to open a tunnel to the database that is owned by this account.

The table below lists the tasks and the person responsible for providing access to the database in another account:

Table 1:
Task Responsible Commands User
Giving Other Accounts Permission to Open a Database Tunnel

Administrator in the account that owns the database

grant-db-tunnel-access
Opening Tunnels to Databases in Other Accounts

Member of the account that has requested permission to open a tunnel to a database owned by another account

 open-db-tunnel
Revoking Tunnel Access to Databases for Other Accounts

Administrator in the account that owns the database

 revoke-db-tunnel-access
