Integration with the Application

Integration with the application covers the technical integration of the target application with the APIs of the gamification service. Firstly, integration is required to send events that are of interest to the gamification service, for example to send the event that a user in a call center has successfully processed a ticket. Secondly, integration is necessary to notify the user about his/her achievements, to send notifications to the user for earned points, or to display the userâ€™s profile.

The gamification service is designed to support the integration of mainly cloud applications running with SAP Cloud Platform. Integration of other applications is technically possible, but restricted for security reasons.
