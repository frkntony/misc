Updating SAP Development Tools for Eclipse

Context

If you have already installed SAP Cloud Platform Tools, you only need to update them. To do so, follow the steps below.

Procedure
  1. Ensure that the SAP Cloud Platform Tools software site is checked for updates:
    1. Find out whether you are using a Mars or Neon release of Eclipse. The name of the release is shown on the welcome screen when the Eclipse IDE is started.
      Caution The support for Luna has entered end of maintenance. We recommend that you use Mars or Neon releases.
    2. In the main menu, choose  Window  Preferences  Install/Update  Available Software Sites .
    3. Make sure there is an entry https://tools.hana.ondemand.com/mars or https://tools.hana.ondemand.com/neon and that this entry is selected.
    4. Choose OK to close the Preferences dialog box.
  2. Choose  Help  Check for Updates .
  3. Choose Finish to start installing the updates.
Note If you want to have your SAP Cloud Platform Tools updated regularly and automatically, open the Preferences window again and choose  Install/Update  Automatic Updates . Select Automatically find new updates and notify me and choose Apply.
