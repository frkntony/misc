list-db-tunnel-access-grants

This command lists all current database access permissions for databases in other accounts.

neo list-db-tunnel-access-grants -h <host> -u <user> -a <my account>
Note The list does not include access permissions that have been revoked.
Parameters
Optional
-i, --id (Optional) Database ID

Lists only the access permissions for the specified database

Type: string
Example
neo list-db-tunnel-access-grants -a myaccount -h hanatrial.ondemand.com -u mymail@example.com -i mydb
