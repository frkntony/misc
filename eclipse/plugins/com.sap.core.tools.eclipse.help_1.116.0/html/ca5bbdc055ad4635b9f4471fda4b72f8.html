Defining the Consequences ("THEN")

The declaration of the consequences (“then”) part supports writing code with the Drools Rules Language (DRL) in version 5.6.0 and Java code.

Note

The formatting in the consequences part must be in the Java style. The DRL can be used in combination with Java code.

The consequences part defines what will be executed once the trigger part is fulfilled. It allows you to update the player achievements or to create new events. Multiple consequences can be defined within one consequences part.
