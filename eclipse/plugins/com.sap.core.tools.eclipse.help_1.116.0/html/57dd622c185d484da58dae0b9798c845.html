Custom messages can usually be specified using an optional parameter <notificationMessage> of the corresponding API method.

Examples:

  • addBadgeToPlayer(String playerId, String badgeName, String <notificationMessage>)
  • givePointsToPlayer(String playerId, String pointType, double amount, String <notificationMessage>)
