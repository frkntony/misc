Updating Rules

Prerequisites

You have logged on to the gamification workbench with the role GamificationDesigner and you have opened the Rules tab.

Procedure

  1. Click on the name of the rule in the entity list to open the rule editor.
  2. Change the rule code.
  3. Press Save.
  4. Optional: Create or modify additional rules.
  5. Close the rule editor and apply changes to deploy the rules.
    Caution A modified rule is not automatically deployed. The deployment is initiated once you have pressed Apply Changes in the rules overview. The rule must be enabled to be deployed.
