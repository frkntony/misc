Translating Individual Files

You use translation projects to translate files of the supported formats as part of the translation workflow of SAP Translation Hub.

Prerequisites

There is a translation project for your source file, containing the texts to be translated.

Context

On the SAP Translation Hub UI (<base URL of SAP Translation Hub>/ui), choose the translation project that refers to the source file to be translated, and choose Get Translations.

Results

The source texts are translated into the target languages specified in the translation project. To check the translations in the project, choose the Translations tab. You see all translations in the translation project and the translation provider that SAP Translation Hub uses for each text.
You also see the quality index, which helps you to judge the quality of translations. The higher the number on a scale from 0-100, the better the quality. The quality index is based on the following:
  • Ratings that language specialists assign to translations
  • The number of instances in the multilingual text repository in which the combination of a source text and a particular translation occurs. The more often a given combination of source text and translation occurs, the more confident we are that the translation is good.
Note The quality index currently appears for translations that are provided by the multilingual text repository only.
