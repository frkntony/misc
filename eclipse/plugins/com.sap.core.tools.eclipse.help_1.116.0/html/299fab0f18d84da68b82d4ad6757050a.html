Opening SAP River RDE

Open SAP RDE in a supported web browser.

Prerequisites

Ensure you have configured SAP RDE as described in the previous sections.

Procedure

  1. Click on the subscription you created for SAP RDE.

    See Subscribing to the SAP River RDE.

    You get an application URL.
  2. Click on the URL or copy it into your browser.
    Note The latest version of both Google Chrome and Firefox are currently supported.
Related Information
For detailed documentation on SAP River RDE, go to HANA Cloud Portal Documentation at https://help.hana.ondemand.com. Under Platform Solutions, choose SAP River Rapid Development Environment.
