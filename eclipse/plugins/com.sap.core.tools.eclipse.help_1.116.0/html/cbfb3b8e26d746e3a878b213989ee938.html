Samples

You can easily import SAP HANA XS sample applications in your Eclipse IDE and deploy them on your SAP HANA schema and package on SAP Cloud Platform (Productive or Trial landscape).

Prerequisites

  • In your Eclipse IDE, you have installed:

    • Sample Applications for SAP Cloud Platform Feature from the SAP Cloud Platform Tools

    • SAP HANA Tools

    See Installing SAP HANA Tools for Eclipse

  • You have a SAP HANA instance created (In the cockpit, choose Databases & Schemas.)
  • In your Eclipse IDE, you have the SAP HANA system added. See Connecting to SAP HANA Schemas via the Eclipse IDE or Connecting to SAP HANA Databases via the Eclipse IDE

Context

  • Importing the Samples in Eclipse IDE

  • Running the HelloWorld Application

  • Running the SAP HANA Interactive Education (SHINE) Application

Importing the Samples in Eclipse IDE

Procedure

  1. In your Eclipse IDE, choose  File  Import  SAP HANA Content  Sample Applications .
  2. Select the sample application you need and choose Next.
  3. Select the SAP HANA system where you want to add the application and choose Next.
  4. Select the schema and package where you want to add the application and choose Finish.
    Restriction

    Choose a package that does not contain conflicting XSAPP objects. We recommend importing the application into an empty package.

Running the HelloWorld Application

Procedure

  1. In your Eclipse IDE, import the HelloWorld sample.
  2. Grant the <your_package>.helloworld::helloworld_user role to your SAP ID service user.

    Do this in Eclipse IDE. From the SQL console, execute the following statement: 

    call "HCP"."HCP_GRANT_ROLE_TO_USER" ('<your_package>.helloworld::helloworld_user', '<SAP_ID_service_user_name>')

    For example:

    call "HCP"."HCP_GRANT_ROLE_TO_USER" ('p123456789trial.mysamples.helloworld::helloworld_user', 'p123456789')
  3. In the cockpit, go to section HANA XS Applications.
  4. Select your HelloWorld application, and click on the application URL to open it in the Web browser.

Results

In your Web browser, you can now see the simple HelloWorld UI.

Running the SAP HANA Interactive Education (SHINE) Application

SHINE is a demo application that makes it easy to learn how to build SAP HANA applications. The demo application comes complete with sample data and design-time developer objects for the application's database tables, data views, stored procedures, OData, and user interface. The SHINE application is available in two variants: SHINEOpenSAP and SHINESP8.

SHINEOpenSAP

SHINEOpenSAP is a simplified version, especially suitable for the OpenSAP course.

Procedure

To run this sample:

  1. Import the SHINEOpenSAP sample in your Eclipse IDE.
  2. Grant the role <your_package>.shineopensap.data::model_access to your SAP ID service user.
    This is done in Eclipse IDE. From the SQL console, execute the following statement:
    CALL "HCP"."HCP_GRANT_ROLE_TO_USER"('<your_package>.shineopensap.data::model_access', '<SAP_ID_service_user_name>')

    For example:

    CALL "HCP"."HCP_GRANT_ROLE_TO_USER"('p123456789trial.mysamples.shineopensap.data::model_access', 'p123456789')
  3. The SHINE sample application has an administration UI that allows sample data generation. To access the administration UI, execute the following statement in the SQL console:
    CALL "HCP"."HCP_GRANT_ROLE_TO_USER"('<your_package>.shineopensap.data::model_admin', '<SAP_ID_service_user_name>')
  4. In the cockpit, go to section HANA XS Applications.
  5. Select your SHINE application, and click on the application URL to open it in the Web browser.

    To access the admin UI from your SHINE application URL, replace the following part from the URL:

    <shine>/shine/ui/poworklist/WebContent/poWorklist.html

    with

    <shine>/shine/admin/ui/WebContent/admin.html

SHINESP8

SHINESP8 is the full version, suitable for SAP HANA SP8 or higher.

Procedure

To run it:

  1. Import the SHINESP8 sample in your Eclipse IDE.
  2. In Eclipse IDE, execute the following statement in the SQL console:
    CALL "HCP"."HCP_GRANT_ROLE_TO_USER"('<your_package>.shinesp8.roles::User', '<SAP_ID_service_user_name>')

    For example:

    CALL "HCP"."HCP_GRANT_ROLE_TO_USER"('p123456789trial.mysamples.shinesp8.roles::User', 'p123456789')
  3. In the cockpit, go to section HANA XS Applications.
  4. Select your SHINE application, and click on the application URL to open it in the Web browser.

    To access the admin UI from your shine application URL, replace the following part from the URL:

    <shine>/shine/ui/poworklist/WebContent/poWorklist.html

    with

    <shine>/shine/admin/ui/WebContent/admin.html

Results

In your Web browser, you can now see the SHINE application UI. You can generate sample data, create purchase order worklists, see different dashboards and so on.
