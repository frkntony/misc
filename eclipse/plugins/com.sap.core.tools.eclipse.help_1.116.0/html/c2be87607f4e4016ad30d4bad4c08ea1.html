Monitoring Database Systems

To monitor whether your deployed SAP HANA XS application is up and running, you can register an availability check for it and configure email recipients who will receive notification if the application goes down. For the email recipients configuration, you use the SAP Cloud Platform console client. Furthermore, you can just view the metrics of a database system of any type.
