Operator's Guide

After you have developed and deployed your application on SAP Cloud Platform, you can configure and operate it. Operator tasks are enabled by two tools: SAP Cloud Platform Platform Cockpit and SAP Cloud Platform console client, while the Eclipse IDE is only used for development purposes.

In addition, you can optionally use the SAP Cloud Platform feedback service to get feedback for your applications from users. You can also analyze feedback responses you get and eventually improve the appearance, performance, and user experience of your applications.
