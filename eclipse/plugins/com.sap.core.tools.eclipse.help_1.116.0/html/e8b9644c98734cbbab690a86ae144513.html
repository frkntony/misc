Using the Notification Mechanism

Notifications are messages that inform users about certain state changes, for example earned achievements, new missions, new teams. They are considered "see and forget" information and won't stay long in the system.

Context

On one hand, notifications are created automatically when calling certain API methods. On the other hand, you can also create and assign custom notifications by using the methods addCustomNotificationToPlayer and addCustomNotificationToTeamMembers.

Notifications are delivered to players or teams by implementing a polling-based approach using the API methods getNotificationsForPlayer and getAllNotifications.
