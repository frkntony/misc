Create Destination for Getting the Player's Achievements

Procedure

  1. Enter the name: “gswidgetdest”.
  2. Select the type: “HTTP”.
  3. Optional: Enter a description.
  4. Enter the URL of the API of your subscription: https://<Application URL>/gamification/api/user/JsonRPC

    For example: https://<yoursubscription>.hana.ondemand.com/gamification/api/user/JsonRPC.

    You can find the Application URL of your service instance by navigating to the gamification workbench ( Account  Services  Go to Service )

  5. Select the proxy type: “Internet”.
  6. Select the cloud connector version: “2”.
  7. Select the authentication: “AppToAppSSO”
  8. Press Save.
