Navigation with Multiple Apps Inside the Launchpad

Describes how navigation with multiple apps inside the launchpad works

The shell uses part of the hash for itself. A URL can look like this:

http://veui5infra.dhcp.wdf.sap.corp:8080/demokit/#shellInformationLikeAppName&/navigationHash/3/detail/1

You can use the navigation framework either with the shell, or standalone. You do not have to take care of this as it is already handled within the framework. Only for app-to-app navigation you have to implement the required information.
