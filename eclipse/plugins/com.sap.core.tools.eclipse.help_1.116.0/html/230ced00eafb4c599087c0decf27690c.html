Connecting to a Repository (Java)

Your application must be connected to the repository you created.

Context

There are many ways to connect to a repository. For more information, see the API Documentation and Reuse OpenCmis Session Objects in Performance Tips (Java).

Procedure

To connect to an existing repository, use the connect(uniqueName, key) method.

Once you are connected to the repository, you get an OpenCMIS session object to manage documents and folders in the connected repository.
