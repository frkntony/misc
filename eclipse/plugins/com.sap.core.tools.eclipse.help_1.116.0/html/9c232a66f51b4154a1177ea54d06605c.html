Creating a Project and Adding an HTML File

A project is needed to create files and to make them available in the cockpit.

Procedure

  1. Create a project.
    1. In the Git repository view, right click on the repository and select Import Projects.
    2. Choose Import as general project.
    3. In the Project Name field, ensure that the project name is shown, and choose Next.
    4. Enter a name for your project.
    5. Choose Finish.
  2. Add an HTML file.
    1. In the JavaScript or the Resource perspective of Eclipse, start the wizard by right-clicking your project and choosing  New  File .
    2. Enter the following name for your file: index.html.
    3. Choose Finish.
    4. In your new index.html file, enter, for example, the Hello World code snippet: 
      <html>
<body>
	<h1>Hello World</h1>
</body>
</html>
    5. Save your entries.
