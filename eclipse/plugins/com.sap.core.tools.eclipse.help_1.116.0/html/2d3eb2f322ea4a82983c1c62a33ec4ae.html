Variant for Bootstrapping from SAP Cloud Platform

An alternative variant for bootstrapping is supported when using SAPUI5 from SAP Cloud Platform.

You can use SAPUI5 in any HTML page from SAP Cloud Platform with the following bootstrap script:

<script id="sap-ui-bootstrap"
    type="text/javascript"
    src="https://sapui5.hana.ondemand.com/resources/sap-ui-core.js"
    data-sap-ui-theme="sap_bluecrystal"
    data-sap-ui-libs="sap.m"></script>

If you want to make sure that you refer to a stable version of SAPUI5, you can use a versioned URL as in the example below:

<script id="sap-ui-bootstrap"
    type="text/javascript"
    src="https://sapui5.hana.ondemand.com/1.28.8/resources/sap-ui-core.js"
    data-sap-ui-theme="sap_bluecrystal"
    data-sap-ui-libs="sap.m"></script>

The first segment of the URL after the host name is used to specify a concrete SAPUI5 version.

The available SAPUI5 versions with respective maintenance status are listed in the https://sapui5.hana.ondemand.com/versionoverview.html in the Demo Kit.

The SAPUI5 maintenance strategy for SAP Cloud Platform is available in this document.
