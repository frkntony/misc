Configure Destinations

Prerequisites

You have logged into the SAP Cloud Platform cockpit with your SCN user and password.

Context
You need to configure a destination to allow the communication between your application (in this case, a sample app) and your subscription to the gamification service. For the sample application, two destinations are necessary:
  • For the gamification service API to send the events: gsdest
  • For the gamification service API to request the achievements: gswidgetdest
Note

The destinations must be created on account level for your personal development account.
