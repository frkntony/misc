Create Destinations

You need to configure a destination to allow communication between your application (in this case the sample app) and your subscription of the gamification service. For the sample app, two destinations are necessary:

  • The destination of the gamification service API to send the events: gsdest

  • The destination of the gamification service API to request the achievements: gswidgetdest

For details about configuring destinations, see Configuring Destinations from the Cockpit

Prerequisite

Configure your account to allow principal propagation. For more information, see HTTP Destinations

Note The destinations must be created on account level.
