list-mta-operations

This command shows the MTA operation status with a given ID.

neo list-mta-operations --host <landscape_host> --account <account_name> --operation-id < operation-id > --user <e-mail_or_user>
Parameters
To list all parameters available for this command, execute neo help list-mta-operations in the command line.
Required
-a, --account The name of the account for which you provide a user and a password.
-h, --host The landscape host on which you execute the command.
-p, --password Your user password. We recommend that you enter it only when prompted, and not explicitly as a parameter in a properties file or the command line.
-u, --user Your user email or SAP ID (SCN) user name.
--operation-id The ID of the operation.
Note This parameter is optional. If you do not use this parameter, all operations that have not been cleaned up within the last 24 hours will be listed.
Example
To check the deployment status of an MTA with ID "i2", execute:
neo list-mta-operations –host hana.ondemand.com --account myacc –-operation-id i2 --user mymail@example.com
