Create Destination for Sending Events

Procedure

  1. Enter the name: “gsdest”.
  2. Select the type: “HTTP”.
  3. Optional: Enter a description.
  4. Enter the URL of the API for your subscription: https://<Application URL>/gamification/api/tech/JsonRPC.

    https://<yoursubscription>.hana.ondemand.com/gamification/api/tech/JsonRPC .

    You can find the Application URL of your service instance by navigating to the gamification workbench ( Account  Services  Go to Service )

  5. Select the proxy type: “Internet”.
  6. Select the cloud connector version: “2”.
  7. Select the authentication: “Basic Authentication”
  8. Enter the technical user. (The technical user must have the “AppStandard” and “AppAdmin” roles.)
  9. Enter the technical user password.
  10. Press Save.
