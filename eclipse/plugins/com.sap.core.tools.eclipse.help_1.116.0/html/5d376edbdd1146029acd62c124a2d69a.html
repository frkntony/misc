Entering the Git User Settings

To commit changes in any SAP River RDE project, your user name and email address must be set for your Git account.

Context

With the following steps you either set the Git user settings or simply check whether they are already set correctly.

Procedure

  1. Open SAP River RDE in a Google Chrome or Firefox browser using the subscription URL you defined.
  2. Switch to the Git view using  Tools  Git Settings .
  3. Enter the email address and full name that are stored for your SCN user.
  4. Choose Update.
